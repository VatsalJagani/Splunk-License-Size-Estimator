Important Notes

Additional Application Logs

Some application logs can vary from minimum size to Terabytes depending on the environment size. These should be measured separately:

  • Cloudflare: minimum 20 GB
  • Custom Applications: minimum 5-10 GB

Cloud Tenancy Logs

Cloud tenancy log sizes are typically small and not included in the calculation above. However, if your estimated license is below 20 GB, you should separately account for:

  • EntraID/Office365
  • Google Workspace
  • AWS CloudTrail
  • Tenable
  • Qualys
  • And other cloud services
